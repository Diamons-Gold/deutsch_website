Die Internetgefahren ( Teil 1 ) :

In Internet, können wir ein Opfer von Hacking sein. Die Hacker können dein Geld oder deine Daten stehlen. es ist viel gefährlich. Um dieses Problem zu nicht haben, es ist wichtig, seine persönliche Daten zu schützen. Ihr können herunterladen ein Antivirus um deinen Computer zu schützen.

Die Internetgefahren ( Teil 2 ) :

Im internet, Meenschen können sich als jemand anderes ausgeben. Im Sozial Netzwerke, ist es viel üblich. Wenn eine Person, die du nicht kennst, frag dich zu treffen, du würde sich weigern weil ist es gefährlich. Sie können a Identität eines anderen nehmen um die Person zu belästigen.

Die Internetgefahren ( Teil 3 ) :

Es gibt auch das Problem der Belästigung im Internet und meist im Sozial Netzwerke. der Belästigung ist eine schreckliche Tat, die kann zum Selbstmord gehen. Wenn Sie belästigt werden, müssen Sie es unbedingt sagen. Und sie sind Zeuge dieser Tat, du musst es denunzieren.

"Internet" eine Gefahr ( Teil 1 ) :

Das Internet hat seine Gefahren aber „Internet“ ist auch eine Gefahr. Wir können ein Smombies werden. Sein ein Smombies, ist das abhängig von seinem Handy oder von seinem Computer sein

"Internet" eine Gefahr ( Teil 2 ) :

Sein ein Smombies ist das nicht cool. Die virtuelle Welt zerstört die wahre Welt und die virtuellen Freunde ersetzen echte Freunde. Der Smombies isoliert sich und spricht nich mehr. Der Smombies lebt nicht mehr in die wahre Welt.

"Internet" eine Gefahr ( Teil 3 ) :

Die Internetabhängigkeit ist manchmal so stark, dass die Smombies nicht mehr essen. Es ist traurig, so zu enden und es ist sehr ernst. Am Ende müssen die am meisten süchtigen Smombies entgiftet werden.
